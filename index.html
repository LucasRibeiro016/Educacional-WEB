DevMini

Sobre

O DevMini é um projeto educativo voltado para iniciantes que querem aprender HTML e CSS de forma simples e prática. O site apresenta desde conceitos básicos de como a web funciona até a construção e estilização de páginas. O conteúdo inclui exemplos claros e exercícios práticos para facilitar o aprendizado. O objetivo é incentivar a prática, autonomia e curiosidade de quem quer começar no desenvolvimento web. O projeto faz parte de uma atividade de extensão universitária, oferecendo aprendizado real à comunidade. É totalmente online e acessível a qualquer pessoa, sem necessidade de experiência prévia.
